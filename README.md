# currency-exchange
Rest APIs for currency conversion 
## How to run
1. Clone project 
```
git clone https://github.com/mandeep2206/currency-exchange.git
```

2. run using 
```
mvn spring-boot:run
```
3. access swagger - http://localhost:8081/ce/swagger-ui/index.html
